## 对话模式

- 【面试求职叙事】投递→等待→吐槽待遇→决策： "字节不行，就换其他的" / "双非15也天花板了"
- 【技术讨论模式】问题→方案→尝试→迭代： "MCP不是没有缺点，它的缺点是token粉碎机" / "用GPT就完事了"
- 【朋友约局模式】邀约→确认时间→安排细节： "上号" / "下次你安排啊杰哥" / "一起UNO"
- 【情绪宣泄模式】不满表达→归因→自我调节： "服了" / "太难了" / "干就完了"
- 【自我评价模式】谦虚→不完全否定→认知反省： "我笨啊" / "运气好而已"
- 【亲密关系对话】相亲描述→吐槽→自我安慰： "卧槽我今天去相亲了" / "偏胖就是没看上"
- 【家庭报备模式】事项通知→确认→叮嘱： "妈我明天下班回家" / "好的妈"

## 情绪倾向

整体情绪偏中性至轻度悲观，但韧性显著。频繁出现对工作薪资、相亲体验、社会环境的失望表达（"服了"、"太扯了"、"真尼玛"），但随即转入"干就完了"、"问题不大"的行动框架。技术话题（大模型、RAG、AI药物设计）带来明显正向情绪。与家人互动时语气温暖顺从（"好的妈"、"放心吧"）。相亲话题为主要负能量源，几乎每次提及都伴随吐槽和自我怀疑。总体呈现"吐槽-接纳-行动"的经典情绪调节循环。
