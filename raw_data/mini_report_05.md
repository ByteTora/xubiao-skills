## 人格/价值观片段

- **自我评价**: "咱就是外在差点，其他我个人觉得还行，也努力上进，脑子也聪明" — 自信
- **恋爱底线**: "不能当舔狗" / "不是钱的事儿，我有钱没钱都不会当舔狗" — 原则清晰
- **养老观念**: "尤其像咱这一代，更清醒了" — 代际对比
- **婚恋观**: "找了对象就要结婚，结了婚就要生孩子，没有消停时候" — 清醒认知
- **金钱观**: "没了再挣" / "挣钱就是用来花的，留着干啥!" — 豁达消费
- **家庭关系**: "对咱妈拿出一部分彩礼置办陪嫁什么意见" — 重视家人意见
- **自我坚持**: "我不想减肥" / "你自己是最重要的" — 自我优先
- **职业态度**: "先别脱产啊" — 务实
- **相亲态度**: "实话实说就行，看不上就算了，没缘分" — 干脆利落
- **人际**: "自己人要宽容，相亲的女的，去他妈的" — 内外有别

- **找房/求职双线**: "我准备去北京了行哥" / "问题不大，弄" — 务实规划

## 情绪倾向

情感波动较batch_04更大，出现更多负面情绪（"真服了"、"烦死了"、"恶心"、"裂开"）。但低落情绪仍被"没事"、"问题不大"等快速中和。相亲话题带来明显挫折感（"不想再相了妈"），但整体保持积极行动导向，频繁出现"搞钱"、"安排"等主动词汇。
