## 对话模式

- 【技术协作模式】问题确认→调试→验证→完成："nvidia-smi 看下输出" / "按理说ollama run qwen3:8b 直接就起来了啊"
- 【求职面试模式】投递→等待→面经分享→决策："刚面了一个" / "感觉还行，先看看说" / "又分手了姐们"—offer rejection类比dating
- 【家庭安居模式】父母→汇报→叮嘱→确认："住到了妈" / "安全的到济南了" / "到家啦到家啦"
- 【亲密关系模式】见面邀约→心理博弈→自嘲收尾： "只是想多接触一下" / "我觉得是有的" / "不聊了哥哥，睡觉了"
- 【职场吐槽模式】不满→归因→应对："不能干外包" / "干不过资本啊" / "准备跑路了"
- 【自我成长模式】反思→顿悟→行动："我发现我不能自己憋着" / "技多不压身" / "学会了沉没成本不参与重大决策"
- 【朋友约局模式】"上号上号" → 时间确认 → 安排地点

## 情绪倾向

情绪中位偏正向，相比前两批更稳定。技术话题（AI、大模型、编程工具）带来明显正情绪驱动；求职话题从焦虑转向自信化（"offer有的是"、"我嫌工资低"）。幽默调侃比例升高（"笑死"、"直接王炸"）。家庭话题持续温暖。创业和技术突破话题带来乐观情绪峰值。负面情绪集中在相亲吐槽和职场不满，但严重程度降低，"问题不大"使用频率持续高位，反映情绪调节成熟度提升。
